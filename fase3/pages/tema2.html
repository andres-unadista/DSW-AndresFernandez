Frameworks CSS y Posicionamiento Web

Posicionamiento WEB

Estructura recomendada

El tipo de estructura que suele ser más beneficiosa para el SEO es la estructura "silo". Este tipo de estructura en forma de árbol o pirámide te permite encaminar más fácilmente a los usuarios y llevarlos hacia donde más te interese, como por ejemplo las "Landing Pages" donde quieres que se lleve a cabo alguna acción (compra, suscripción, etc.)(Consejos HTML para mejorar tu SEO, s. f.)

El manejar un buen SEO en los sitios permite que sean mejor ranqueados en los buscadores como Google, esto permite que aumente el número de visitantes. Lo anterior es beneficioso para cualquier sitio, por lo tanto, es bueno hacer uso de las etiquetas y rellenar sus propiedades. Se recomienda:

  • Tratar que cada página del sitio contenga información unica
  • Manejar correctamente las etiquetas de títulos h1, h2, h3, h4
  • Tratar que los títulos tengan varias palabras clave
